Meus Projetos

ChatBot - OuviTexto

O projeto social OuviTexto tem como objetivo ajudar pessoas não alfabetizadas a entender textos contidos em imagens, banners, cartazes e outros materiais. Através de um chat interativo, o usuário poderá enviar uma mensagem de áudio acompanhada de uma imagem, solicitando a descrição do texto presente na imagem. Em resposta, o sistema enviará uma mensagem de áudio explicando o conteúdo do texto na imagem, de forma acessível e compreensível. O desenvolvimento do projeto surgiu a partir da necessidade de apoiar a inclusão digital e social de pessoas em situação de vulnerabilidade, fornecendo uma ferramenta que facilita o entendimento de informações visuais e promove a alfabetização digital. O OuviTexto visa ser uma solução prática e inovadora, utilizando tecnologias de processamento de imagem e áudio para atender às necessidades dessa população.

API de classificação de preços de reservas de hotéis

O projeto tem como objetivo treinar um modelo de aprendizado de máquina para classificar reservas de hotel em três faixas de preços, utilizando o conjunto de dados "Reservas de Hotel". O modelo foi treinado com o AWS SageMaker e a API desenvolvida com FastAPI permite que os usuários enviem dados de reservas e recebam a faixa de preço correspondente. A API expõe um endpoint /api/v1/inferencepara realizar inferências com o modelo salvo no S3.
